# Aloha Window Bros — Site Map

## Tech Stack
- **Framework**: Astro 6 (static output)
- **Styling**: Global CSS (Inter + Playfair Display fonts), blue→purple wave theme from truck branding
- **Content**: Astro Content Collections (markdown with frontmatter)
- **Tracking**: GA4 (G-MRSP8D9CQG) with custom conversion events
- **Deploy**: Docker nginx container via blrb-connect, GitHub Actions auto-deploy

## SEO
- Per-page title, description, canonical, Open Graph
- JSON-LD: LocalBusiness (home), Article (blog), Service (services)
- Sitemap at /sitemap-index.xml
- 301 redirects from old WordPress URLs
